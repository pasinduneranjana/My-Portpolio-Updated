# Pasindu Naranjana | Personal Portfolio

A sleek, modern, dark-themed personal portfolio showcasing my expertise in **Network Engineering** and **Cloud Computing**. Designed with a focus on performance, aesthetics, and user experience.

![Portfolio Preview](./public/vite.svg) <!-- Replace with a real screenshot if available -->

## 🚀 Features

- **Modern Glassmorphism UI**: High-end visuals using Tailwind CSS with glass-like components.
- **Dark Mode Only**: Built from the ground up to support a consistent, deep, and immersive dark theme.
- **Smooth Animations**: Interactive scroll animations and micro-interactions powered by Framer Motion.
- **Fully Responsive**: Optimized for seamless viewing on desktops, tablets, and mobile devices.
- **Functional Contact Form**: Integrated with EmailJS for direct communication.

## 🛠️ Technologies & Tools

- **Frontend**: [React 19](https://react.dev/), [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Email Service**: [EmailJS](https://www.emailjs.com/)

## 💻 Getting Started

Follow these steps to run the portfolio locally on your machine.

### Prerequisites
- Node.js (v18 or higher recommended)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/pasinduneranjana/pasindu-portpolio.git
   cd pasindu-portpolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up EmailJS (Optional but recommended)**
   If you want the contact form to work, create a `.env` file in the root directory (if configured) or update the keys directly in `src/components/Contact.jsx` with your EmailJS credentials:
   ```javascript
   const SERVICE_ID = "your_service_id";
   const TEMPLATE_ID = "your_template_id";
   const PUBLIC_KEY = "your_public_key";
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```
